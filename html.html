Introdução ao HTML

HTML (HyperText Markup Language) é a linguagem usada para criar a estrutura de páginas web. É a base de todo conteúdo da web.

Estrutura Básica de uma Página HTML

Uma página HTML é composta por diversas tags que definem seu conteúdo. Aqui está um exemplo básico:

<!DOCTYPE html>
<html>
    <head>
        <title>Minha Página</title>
    </head>
    <body>
        <h1>Olá, Mundo!</h1>
    </body>
</html>

Principais Tags e Elementos

  • <p>: Parágrafo de texto.
  • <h1> a <h6>: Títulos de diferentes níveis.
  • <a>: Link para outra página.
  • <img>: Imagem.
  • <ul> e <li>: Lista não ordenada.
  • <table>: Tabela.

Exercício Prático

Tente criar seu próprio documento HTML com o seguinte conteúdo:

<!DOCTYPE html>
<html>
    <head>
        <title>Minha Primeira Página</title>
    </head>
    <body>
        <h1>Bem-vindo ao HTML!</h1>
        <p>Este é um parágrafo de exemplo.</p>
    </body>
</html>

Abra o seu editor de texto (como o Notepad) e salve o código acima como pagina.html. Em seguida, abra o arquivo no seu navegador.
