Introdução ao CSS

CSS (Cascading Style Sheets) é a linguagem usada para estilizar páginas HTML. Com ele, você pode controlar cores, fontes, espaçamentos e muito mais.

Como Aplicar CSS em um Documento HTML

Existem três maneiras de adicionar CSS a um documento HTML:

  • Inline: Usando o atributo style diretamente nos elementos HTML.
  • Interno: Adicionando um bloco <style> dentro da tag <head>.
  • Externo: Usando um arquivo CSS separado.
/* Exemplo de estilo interno */
<style>
    p {
        color: blue;
    }
</style>

Seletores e Propriedades

CSS usa seletores para aplicar estilos a elementos. Aqui estão alguns exemplos:

  • Seletores de tag: p, h1, etc.
  • Seletores de classe: .minha-classe
  • Seletores de ID: #meu-id
/* Exemplo de uso */
p {
    color: green;
}
.minha-classe {
    background-color: yellow;
}
#meu-id {
    font-size: 20px;
}

Exercício Prático

Tente criar um arquivo HTML e um arquivo CSS para estilizar uma página. Use o código abaixo como referência:

<!-- Arquivo HTML (index.html) -->
<!DOCTYPE html>
<html>
<head>
    <title>Minha Página com CSS</title>
    <link rel="stylesheet" href="estilos.css">
</head>
<body>
    <h1>Bem-vindo ao CSS!</h1>
    <p class="destaque">Este é um parágrafo estilizado.</p>
    <p id="importante">Este é um parágrafo importante.</p>
</body>
</html>
    
/* Arquivo CSS (estilos.css) */
body {
background-color: #f0f8ff;
}
h1 {
color: #0066cc;
}
.destaque {
color: #ff6347;
font-weight: bold;
}
#importante {
color: #2e8b57;
text-decoration: underline;
}

Salve o HTML como index.html e o CSS como estilos.css na mesma pasta. Abra o arquivo HTML no seu navegador para ver o resultado.
