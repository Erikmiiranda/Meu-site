Introdução ao JavaScript

JavaScript é uma linguagem de programação usada para tornar páginas web interativas. Com ele, você pode responder a eventos do usuário, validar formulários, criar animações e muito mais.

Como Inserir JavaScript em uma Página HTML

Existem três maneiras principais de adicionar JavaScript a um documento HTML:

  • Inline: Usando o atributo onclick ou outras propriedades diretamente nos elementos HTML.
  • Interno: Adicionando um bloco de <script> dentro da página.
  • Externo: Linkando um arquivo JavaScript separado usando <script src="script.js">.
<!-- Exemplo de script interno -->
<script>
    console.log("Olá, Mundo!");
</script>

Manipulação de Elementos e Eventos

Você pode usar JavaScript para manipular elementos da página e responder a eventos do usuário, como cliques em botões. Por exemplo:

<button onclick="mostrarMensagem()">Clique aqui</button>

<script>
function mostrarMensagem() {
    alert("Você clicou no botão!");
}
</script>
